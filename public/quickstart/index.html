This Vagrant file will install a WHMCS compatible LAMP stack on top of Ubuntu. You can spin up and tear down your machine as often as you want to, without having to worry about ionCube and other dependencies to run WHMCS.

Below you will find connection details for the Vagrant box to allow you to connect to the MySQL server from both WHMCS and a local client.

Getting Started #back to top

You will first need to download a copy of WHMCS and place it into the /public/whmcs directory. Once you've done that, you can navigate to it and follow the installation guide, using the MySQL credentials below.

MySQL Connection Details

Connecting from WHMCS (PHP)

Server Hostname localhost
Port 3306 (default)
Username whmcs
Password password
Database name whmcs_default

Connecting from a MySQL client (Sequel Pro / MySQL Workbench)

Before you can connect to MySQL, you will need to connect to the Vagrant box via an SSH tunnel. Run vagrant ssh-config from terminal to view your SSH details. You can then input those credentials into your MySQL client, along with the following details to connect to the MySQL server.

Connecting via a MySQL client makes it easy for you to manage your databases. It's like phpMyAdmin on roids.

Server Hostname 127.0.0.1
Port 3306 (default)
Username whmcs
Password password
Database name whmcs_default or leave blank
